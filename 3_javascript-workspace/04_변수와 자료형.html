변수와 자료형

* 변수 선언 시 var 뿐만 아니라 let, const도 사용 가능

var, let, const의 차이점

1) 변수 선언 방식

2) 스코프 (변수 사용 가능 영역)

* 변수 선언 위치에 따른 전역변수 / 지역변수

        <script>

            var 변수명; // 전역변수

            변수명; // 전역변수

            function 함수명() {
                var 변수명; // 지역변수

                변수명; //  
            }

        </script>
