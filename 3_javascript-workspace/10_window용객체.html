window용 객체

        window 객체는 자바스크립트의 최상위 객체 크게 BOM DOM으로 나뉨
        - BOM (Browser Object Model) : 브라우저 창을 핸들링 가능
                                        (window, location, screen, navigator, history)
        - DOM (Document Object Model) : 요소들을 핸들링 가능
                                        (document)

- window.open('url', '창의 이름', '창의 특성');
